# Seamless Migration of the practice to Technijian Datacenter 3CX

- **Industry:** Healthcare
- **Service theme:** voip-telephony
- **Engagement date:** 2024-07-02
- **Estimated effort:** 47 hrs (1232: 47h)

## Client context

Technijian proposes a seamless migration of the practice from their current 3CX system, to which we do not have admin access, to a new 3CX system hosted in the Technijian Datacenter. T

## Scope of work

### Planning and Preparation

**Problem / context:**

In this phase, we will plan and prepare for the migration by assessing the current system and setting up the necessary infrastructure in the Technijian Datacenter.

**Strategy:**

Conduct a comprehensive assessment of the current 3CX system.Design the new 3CX system architecture in the Technijian Datacenter.Develop a migration plan with detailed steps and timeline.

**Objectives:**

Assess current 3CX system setup and configurations.Prepare Technijian Datacenter for new 3CX system deployment.Ensure all prerequisites for migration are met.

**General requirements:**

Coordination with the practice IT staff.Detailed project plan with timeline and milestones.

**Assumptions:**

Current 3CX system details are provided by the practice.Users are available for training on the new soft phone applications.Access to necessary hardware and network infrastructure.

**Success criteria:**

Review and approval of the migration plan by all stakeholders.

### Installation of Soft Phones

**Problem / context:**

This phase involves installing soft phone applications on users' computers and cell phones to ensure they can continue to make and receive calls during the migration.

**Strategy:**

Schedule installation and training sessions with users.Provide step-by-step guidance for soft phone installation.Conduct training sessions to familiarize users with the new applications.

**Objectives:**

Install soft phone applications on all users' devices.Train users on how to use the soft phones effectively.

**General requirements:**

Installation packages for soft phone applications.Training materials and sessions for users.

**Assumptions:**

Users' devices meet the requirements for soft phone installation.Users are available for installation and training sessions.

**Success criteria:**

Verify successful installation and functionality of soft phones on all devices.

### Porting of DIDs and Flashing of Phones

**Problem / context:**

In this phase, we will port the 2 DIDs to the new 3CX system and flash the 7 physical phones.

**Strategy:**

Coordinate with the telecom provider to schedule and execute the DID porting.Flash the physical phones to the new 3CX system configuration after porting is complete.

**Objectives:**

Successfully port the 2 DIDs to the new 3CX system.Flash and configure the 7 physical phones for the new system.

**General requirements:**

Coordination with the telecom provider for DID porting.Tools and software required for flashing the phones.

**Assumptions:**

Necessary permissions and approvals for porting DIDs are obtained.Physical phones are compatible with the new 3CX system.

**Success criteria:**

Test and verify the functionality of ported DIDs and flashed phones.

### Testing and Validation

**Problem / context:**

This phase involves testing the new system to ensure it functions correctly and validating that all configurations and settings meet the practice’s requirements.

**Strategy:**

Develop and execute test plans covering all aspects of the new system.Collect feedback from users and make necessary adjustments.

**Objectives:**

Conduct comprehensive testing of the new 3CX system.Validate system performance and user satisfaction.

**General requirements:**

Test plans and scripts.Issue tracking and resolution system.

**Assumptions:**

Users are available to participate in testing.All required configurations are completed.

**Success criteria:**

Ensure all tests are passed and issues resolved before final deployment.

### Go-Live and Post-Migration Support

**Problem / context:**

The final phase involves going live with the new 3CX system and providing post-migration support to address any issues that arise.

**Strategy:**

Execute the go-live plan, ensuring all users are transitioned smoothly.Provide dedicated support to address any immediate issues.

**Objectives:**

Successfully transition to the new 3CX system.Provide immediate support to resolve any post-migration issues.

**General requirements:**

Support plan and resources.Communication plan for go-live.

**Assumptions:**

All previous phases are successfully completed.Users are ready to transition to the new system.

**Success criteria:**

Ensure smooth operation of the new system.Confirm user satisfaction and system performance.
