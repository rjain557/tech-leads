## Scope of work

### Preparation and Planning

**Problem / context:**

This phase includes assessing current server configurations, verifying compatibility with Windows Server 2022, and preparing a detailed upgrade plan.

**Strategy:**

Conduct an inventory audit, assess compatibility, and create a migration schedule.

**Objectives:**

To ensure a smooth transition with minimal disruption to current operations.

**General requirements:**

Documented hardware inventory, installed software list, and current configurations for existing systems.

**Assumptions:**

All current hardware meets the minimum requirements for Windows Server 2022 and ESXi 8.0.

### Execution of Upgrade

**Problem / context:**

This phase covers the actual upgrade of five servers to Windows Server 2022 and the transition of two Hyper-V hosts to ESXi 8.0.

**Strategy:**

Implement the upgrade during non-peak hours to minimize operational impact.

**Objectives:**

To successfully upgrade the server systems with limited downtime.

**General requirements:**

Access to all servers for the installation of required software and operating systems.

**Assumptions:**

All necessary backups are completed before the upgrade begins.

### Post-Upgrade Configuration and Testing

**Problem / context:**

This phase includes configuration settings validation, application testing, and troubleshooting any post-upgrade issues.

**Strategy:**

Run tests on all upgraded systems to validate functionality and performance.

**Objectives:**

To ensure all systems are operational and meet the property management company's business needs.

**General requirements:**

Documentation of new system configurations and operational procedures.

**Assumptions:**

All systems are functional after the upgrade and any required data restoration is viable.
