## Scope of work

### Phase 1: Security and Account Management Remediation

**Problem / context:**

This phase will focus on improving the security of user accounts and implementing policies to manage account access more effectively.

**Objectives:**

- Implement a password expiration policy. - Clean up inactive and unnecessary accounts. - Limit admin privileges to essential personnel only.

### Phase 2: System and Endpoint Optimization

**Problem / context:**

This phase concentrates on identifying and addressing issues related to system health and endpoint management.

**Objectives:**

- Reactivate and manage all AD and non-AD computers. - Decommission legacy operating systems.

### Phase 3: Configuration and Organizational Improvement

**Problem / context:**

The focus here is on tidying up configuration settings and organizational structures to improve IT management and performance.

**Objectives:**

- Redistribute FSMO roles for enhanced reliability. - Simplify and clarify Group Policy Objects (GPOs).

### Phase 4: Network and Infrastructure Strengthening

**Problem / context:**

This phase addresses the network infrastructure vulnerabilities and improves overall performance and security.

**Objectives:**

- Rectify network configuration issues and enhance security protocols. - Optimize server storage and manage web servers effectively.

### Phase 5: Peripheral and Application Management

**Problem / context:**

This phase focuses on the management of peripherals and applications to reduce redundancy and improve efficiency.

**Objectives:**

- Evaluate the usage of installed software applications and printers. - Streamline applications to reduce software sprawl.
